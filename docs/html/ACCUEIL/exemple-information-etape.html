Étape 1

Choisir une base de données sur ACLED

ACLED (Armed Conflict Location & Event Data Project) fournit des données détaillées sur les conflits à travers le monde. Nous allons utiliser les données sur l'Irak comme exemple pour ce tutoriel.

Sélection géographique

Choisissez une région spécifique (pays, province) ou un type de conflit particulier pour votre analyse.

Période temporelle

Définissez une période pertinente pour votre étude (année spécifique, décennie, ou tendances récentes).

Filtres avancés

Utilisez les filtres ACLED pour sélectionner des types d'événements, des acteurs spécifiques ou des sous-types de violence.

Téléchargement des données

ACLED propose plusieurs formats de téléchargement. Pour notre analyse, nous utiliserons le format CSV qui est facilement lisible avec pandas. 

# URL de l'API ACLED pour les données sur l'Irak (exemple)
acled_url = "https://api.acleddata.com/acled/read.csv?terms=accept&country=Iraq"

# Téléchargement direct avec pandas
import pandas as pd
df = pd.read_csv(acled_url)

# Aperçu des données
print(df.head())

Étape 2

Traiter les données avec pandas

La bibliothèque pandas offre des outils puissants pour nettoyer, transformer et analyser les données ACLED avant visualisation.

Nettoyage des données

Les données brutes nécessitent souvent un nettoyage pour être exploitables. Voici les opérations essentielles : 

# Conversion des dates
df['event_date'] = pd.to_datetime(df['event_date'])

# Gestion des valeurs manquantes
df['fatalities'] = df['fatalities'].fillna(0)

# Filtrage des colonnes pertinentes
cols_to_keep = ['event_date', 'latitude', 'longitude',
                'event_type', 'sub_event_type', 'fatalities',
               'admin1', 'admin2', 'location']
df = df[cols_to_keep]

# Suppression des doublons
df = df.drop_duplicates()

Analyse exploratoire

Les statistiques descriptives nous aident à comprendre la distribution des données avant toute visualisation. 

# Statistiques de base
print(df.describe())

# Nombre d'événements par type
event_counts = df['event_type'].value_counts()

# Taux de mortalité par région
fatalities_by_region = df.groupby('admin1')['fatalities'].sum().sort_values(ascending=False)

# Évolution temporelle
monthly_events = df.resample('M', on='event_date').size()

Transformation avancée

Pour des analyses plus poussées, nous pouvons créer de nouvelles variables et agrégations. 

# Création de périodes (trimestres)
df['quarter'] = df['event_date'].dt.to_period('Q')

# Classification des événements violents
violent_events = ['Battles', 'Explosions/Remote violence', 'Violence against civilians']
df['is_violent'] = df['event_type'].isin(violent_events)

# Agrégation par région et type d'événement
region_event_matrix = pd.crosstab(
    index=df['admin1'],
    columns=df['event_type'],
    values=df['event_date'],
    aggfunc='count'
)
